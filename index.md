# [3](#i3) <a name='3'></a>. 게임 구성

## [3.1](#i3.1) <a name='3.1'></a>. 메커니즘

[도전 과제]

1) 쏟아져 나오는 적을 죽여라
2) 원하는 전투를 찾아내라

단순히 적들과 싸우고 죽이는 것이 목표.
하지만 중요한건 자신이 원하는 전투 방식을 만들어 내는것이다.

[재미 요소]

1) 적들을 시원시원하게 죽이는 재미
2) 원하는 총을 커스터마이징 하는 재미
적들을 어떻게 죽일까, 어떤 방식이 더 효과적이고 빠를까, 그런것을 상상하며 로드아웃을 빌딩하고, 끝내는
그것을 실전에서 써보며 즐거워 해야 하는것.

[참신함]

# [6](#i6) <a name='6'></a>. 개발 요구사항 & 흐름도

## [6.1](#i6.1) <a name='6.1'></a>. 요구사항
- 시작하면 바로 게임 내부
- ESC를 누르면 메뉴화면
- 메뉴화면은 그래픽, 음량, 키 설정 가능
- 얕은 물로 잠긴 이끼 깔린 바닥과 푸른 하늘, 그리고 하늘을 비추는 얕은 물
- 맵 한가운데를 가로지르는 기차길
- 맵 한 가운데 있는 기차역
- 기차역에 있는 기계
- 기계에서 구매 가능한 아이템
- 기계에서 부를 수 있는 기차
- 기차안은 바깥에서 보는것 보다 넓다
- 기차 칸을 넘어 갈때마다 랜덤한 적이 나온다
- 적을 다 해치우기 전에 문은 열리지 않는다
- 다 해치우면 문이 열린다
- 적을 해치우면 조건 해제로 아이템 부품이 풀린다
- 무기는 총몸 손잡이 개머리판 총열을 조합해 만든다


# [7](#i7) <a name='7'></a>. 구현 1주차 목표 & 작업결과

작업명 : 플레이어

해당 요구사항 : 

1) 1인칭 시점
2) WASD 이동
3) 마우스로 시점 전환
4) SPACE 점프
5) SHIFT 질주
6) 체력바
7) 체력 회복
설명 :

키보드와 마우스를 이용한 1인칭 시점의 움직임을 구현할것.
체력바는 평소에는 보이지 않게하고 캐릭터의 체력이 감소하면 화면에 감소한 체력바를 띄운다.
체력은 일정 시간동안 피해를 입지 않으면 서서히 차오르는것 구현.
체력이 완전히 다 차오르면 일정 시간 후 투명해지며 화면에서 사라진다.
